**–û—Å–Ω–æ–≤–Ω—ã–µ –æ—Ç–ª–∏—á–∏—è –æ—Ç —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–æ–≥–æ –∞–ª–≥–æ—Ä–∏—Ç–º–∞ –ö—Ä—É—Å–∫–∞–ª–∞:**
1. –ü–æ—Å–ª–µ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏—è –∫–ª–µ—Ç–æ–∫, –∞–ª–≥–æ—Ä–∏—Ç–º —Å–ª—É—á–∞–π–Ω—ã–º –æ–±—Ä–∞–∑–æ–º –º–æ–∂–µ—Ç "—Ä–∞–∑—Ä—É—à–∏—Ç—å" —Å—Ç–µ–Ω—ã —Å –Ω–µ–∫–æ—Ç–æ—Ä–æ–π –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å—é (–ø–æ —É–º–æ–ª—á–∞–Ω–∏—é 10%), —Å–æ–∑–¥–∞–≤–∞—è –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã–µ –ø—É—Ç–∏.
2. –ê–ª–≥–æ—Ä–∏—Ç–º —Ç–∞–∫–∂–µ –∏–Ω–∏—Ü–∏–∞–ª–∏–∑–∏—Ä—É–µ—Ç —Ç–∏–ø –ø–æ–≤–µ—Ä—Ö–Ω–æ—Å—Ç–∏ –∫–∞–∂–¥–æ–π –∫–ª–µ—Ç–∫–∏ (–Ω–∞–ø—Ä–∏–º–µ—Ä, –Ω–æ—Ä–º–∞–ª—å–Ω–∞—è, –∑–∞–º–µ–¥–ª—è—é—â–∞—è –∏–ª–∏ —É—Å–∫–æ—Ä—è—é—â–∞—è –ø–æ–≤–µ—Ä—Ö–Ω–æ—Å—Ç—å).

**–®–∞–≥–∏ –∞–ª–≥–æ—Ä–∏—Ç–º–∞:**
1. –ù–∞—á–∞—Ç—å —Å –ª–∞–±–∏—Ä–∏–Ω—Ç–∞, –≥–¥–µ –≤—Å–µ –∫–ª–µ—Ç–∫–∏ –æ–∫—Ä—É–∂–µ–Ω—ã —Å—Ç–µ–Ω–∞–º–∏.
2. –ò–Ω–∏—Ü–∏–∞–ª–∏–∑–∏—Ä–æ–≤–∞—Ç—å —Ç–∏–ø—ã –ø–æ–≤–µ—Ä—Ö–Ω–æ—Å—Ç–µ–π –¥–ª—è –≤—Å–µ—Ö –∫–ª–µ—Ç–æ–∫ —Å–ª—É—á–∞–π–Ω—ã–º –æ–±—Ä–∞–∑–æ–º.
3. –ü–µ—Ä–µ–º–µ—à–∞—Ç—å —Ä—ë–±—Ä–∞.
4. –°–æ–µ–¥–∏–Ω—è—Ç—å –∫–ª–µ—Ç–∫–∏, –∏—Å–ø–æ–ª—å–∑—É—è DSU, –∫–∞–∫ –∏ –≤ –∫–ª–∞—Å—Å–∏—á–µ—Å–∫–æ–º –∞–ª–≥–æ—Ä–∏—Ç–º–µ –ö—Ä—É—Å–∫–∞–ª–∞.
5. –° –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å—é 10% —Ä–∞–∑—Ä—É—à–∞—Ç—å —Å—Ç–µ–Ω—É, –¥–∞–∂–µ –µ—Å–ª–∏ –∫–ª–µ—Ç–∫–∏ —É–∂–µ –Ω–∞—Ö–æ–¥—è—Ç—Å—è –≤ –æ–¥–Ω–æ–π –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–µ, —Å–æ–∑–¥–∞–≤–∞—è –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã–µ –ø—É—Ç–∏.

**–ü—Ä–µ–∏–º—É—â–µ—Å—Ç–≤–∞:**
- –†–∞–∑–Ω—ã–µ —Ç–∏–ø—ã –ø–æ–≤–µ—Ä—Ö–Ω–æ—Å—Ç–µ–π –¥–æ–±–∞–≤–ª—è—é—Ç —ç–ª–µ–º–µ–Ω—Ç —Å—Ç—Ä–∞—Ç–µ–≥–∏–∏ –ø—Ä–∏ —Ä–µ—à–µ–Ω–∏–∏ –ª–∞–±–∏—Ä–∏–Ω—Ç–∞.

**–ü–∞—Ä–∞–º–µ—Ç—Ä—ã:**

## –ê–ª–≥–æ—Ä–∏—Ç–º—ã —Ä–µ—à–µ–Ω–∏—è –ª–∞–±–∏—Ä–∏–Ω—Ç–∞

### 1. –†–µ—à–µ–Ω–∏–µ —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º –ø–æ–∏—Å–∫–∞ –≤ —à–∏—Ä–∏–Ω—É (Breadth-First Search, BFS)

–ê–ª–≥–æ—Ä–∏—Ç–º –ø–æ–∏—Å–∫–∞ –≤ —à–∏—Ä–∏–Ω—É (BFS) ‚Äî —ç—Ç–æ –º–µ—Ç–æ–¥ —Ä–µ—à–µ–Ω–∏—è –ª–∞–±–∏—Ä–∏–Ω—Ç–∞, –∫–æ—Ç–æ—Ä—ã–π –≥–∞—Ä–∞–Ω—Ç–∏—Ä—É–µ—Ç –Ω–∞—Ö–æ–∂–¥–µ–Ω–∏–µ –∫—Ä–∞—Ç—á–∞–π—à–µ–≥–æ (–ø–æ –∫–æ–ª–∏—á–µ—Å—Ç–≤—É –∫–ª–µ—Ç–æ–∫) –ø—É—Ç–∏ –æ—Ç –Ω–∞—á–∞–ª—å–Ω–æ–π –¥–æ –∫–æ–Ω–µ—á–Ω–æ–π —Ç–æ—á–∫–∏. –û–Ω —Ä–∞–±–æ—Ç–∞–µ—Ç –ø—É—Ç–µ–º –ø–æ—Å—Ç–µ–ø–µ–Ω–Ω–æ–≥–æ –∏—Å—Å–ª–µ–¥–æ–≤–∞–Ω–∏—è –≤—Å–µ—Ö —Å–æ—Å–µ–¥–Ω–∏—Ö –∫–ª–µ—Ç–æ–∫ –Ω–∞ –∫–∞–∂–¥–æ–º —É—Ä–æ–≤–Ω–µ, –Ω–∞—á–∏–Ω–∞—è —Å –Ω–∞—á–∞–ª—å–Ω–æ–π –∫–ª–µ—Ç–∫–∏.

**–®–∞–≥–∏ –∞–ª–≥–æ—Ä–∏—Ç–º–∞:**

3. –î–ª—è –∫–∞–∂–¥–æ–π —Å–æ—Å–µ–¥–Ω–µ–π –∫–ª–µ—Ç–∫–∏ –∑–∞–ø–∏—Å—ã–≤–∞—Ç—å, –∫–∞–∫–∞—è –∫–ª–µ—Ç–∫–∞ –ø—Ä–∏–≤–µ–ª–∞ –∫ –Ω–µ–π (–¥–ª—è –≤–æ—Å—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω–∏—è –ø—É—Ç–∏).
4. –ü–æ–≤—Ç–æ—Ä—è—Ç—å –ø—Ä–æ—Ü–µ—Å—Å, –ø–æ–∫–∞ –Ω–µ –±—É–¥–µ—Ç –Ω–∞–π–¥–µ–Ω–∞ –∫–æ–Ω–µ—á–Ω–∞—è –∫–ª–µ—Ç–∫–∞ (`end`) –∏–ª–∏ –ø–æ–∫–∞ –æ—á–µ—Ä–µ–¥—å –Ω–µ —Å—Ç–∞–Ω–µ—Ç –ø—É—Å—Ç–æ–π.
5. –ï—Å–ª–∏ –∫–æ–Ω–µ—á–Ω–∞—è –∫–ª–µ—Ç–∫–∞ –Ω–∞–π–¥–µ–Ω–∞, –≤–æ—Å—Å—Ç–∞–Ω–æ–≤–∏—Ç—å –ø—É—Ç—å, –∏—Å–ø–æ–ª—å–∑—É—è —Å–æ—Ö—Ä–∞–Ω—ë–Ω–Ω—ã–µ —Ä–æ–¥–∏—Ç–µ–ª—å—Å–∫–∏–µ –∫–ª–µ—Ç–∫–∏.

**–ü—Ä–µ–∏–º—É—â–µ—Å—Ç–≤–∞:**
- –ì–∞—Ä–∞–Ω—Ç–∏—Ä—É–µ—Ç –Ω–∞—Ö–æ–∂–¥–µ–Ω–∏–µ –∫—Ä–∞—Ç—á–∞–π—à–µ–≥–æ –ø—É—Ç–∏ –≤ –ª–∞–±–∏—Ä–∏–Ω—Ç–µ –±–µ–∑ –≤–∑–≤–µ—à–µ–Ω–Ω—ã—Ö —Ä—ë–±–µ—Ä.

### 2. –†–µ—à–µ–Ω–∏–µ —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º –∞–ª–≥–æ—Ä–∏—Ç–º–∞ –î–µ–π–∫—Å—Ç—Ä—ã (Dijkstra's Algorithm)

–ê–ª–≥–æ—Ä–∏—Ç–º –î–µ–π–∫—Å—Ç—Ä—ã —Ä–µ—à–∞–µ—Ç –∑–∞–¥–∞—á—É –ø–æ–∏—Å–∫–∞ –∫—Ä–∞—Ç—á–∞–π—à–µ–≥–æ –ø—É—Ç–∏ –≤ –ª–∞–±–∏—Ä–∏–Ω—Ç–µ, –≥–¥–µ —Ä—ë–±—Ä–∞ –º–æ–≥—É—Ç –∏–º–µ—Ç—å —Ä–∞–∑–Ω—É—é —Å—Ç–æ–∏–º–æ—Å—Ç—å (–Ω–∞–ø—Ä–∏–º–µ—Ä, –µ—Å–ª–∏ –µ—Å—Ç—å —Ä–∞–∑–Ω—ã–µ —Ç–∏–ø—ã –ø–æ–≤–µ—Ä—Ö–Ω–æ—Å—Ç–µ–π, –≤–ª–∏—è—é—â–∏–µ –Ω–∞ —Å–∫–æ—Ä–æ—Å—Ç—å –ø—Ä–æ—Ö–æ–∂–¥–µ–Ω–∏—è). –≠—Ç–æ—Ç –∞–ª–≥–æ—Ä–∏—Ç–º –Ω–∞—Ö–æ–¥–∏—Ç –æ–ø—Ç–∏–º–∞–ª—å–Ω—ã–π –ø—É—Ç—å, –º–∏–Ω–∏–º–∏–∑–∏—Ä—É—è –æ–±—â—É—é —Å—Ç–æ–∏–º–æ—Å—Ç—å –ø–µ—Ä–µ–¥–≤–∏–∂–µ–Ω–∏—è –æ—Ç –Ω–∞—á–∞–ª—å–Ω–æ–π –∫–ª–µ—Ç–∫–∏ –∫ –∫–æ–Ω–µ—á–Ω–æ–π.

**–®–∞–≥–∏ –∞–ª–≥–æ—Ä–∏—Ç–º–∞:**
